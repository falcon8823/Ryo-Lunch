= Ryo-Lunch
長野高専 寮食堂の献立配信サービス

== What's this?
* これは長野高専寮食堂の献立配信サービス「Ryo-Lunch」です．

== In future...
* 寮食メニューを写真付きブログの様な形で提供します．
* 寮食を通じたSNSを目指していきます．

== 開発履歴
* 0.1
 * 献立登録機能
 * 空メールによるユーザ登録/解除機能
 * メール自動配信機能
 * 自動Twitter投稿機能
 * 端末に応じたビュー(PC, スマホ, フィーチャーフォン)
 * 管理者パネル
